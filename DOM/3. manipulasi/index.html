Document Manipulasi Element

INI AKAN DIRUBAH

  • innerHTML

    innerHTML bisa digunakan untuk merubah data didalam suatu element, bisa berupa script program/teks. Jika hanya untuk teks lebih disarankan menggunakan textContent agar script program diconvert menjadi string

    contohnya: paragraf.textContent = "ini text"

  • style

    untuk menambahkan inline css, untuk program css lebih dari 1 kata, gunakan camelCase

    contohnya: paragraf.style.backgroundColor = "red"

  • setAttribut

    untuk memanipulasi attribut pada HTML, sebenarnya ada banyak manipulasi atribut seperti:

    1. setAttribute("id","manipulasi")

      untuk menset attribut, seperti membuat attribut baru/ merubah isi suatu atribut. Contohnya membuat/merubah attribut id menjadi "manipulasi"

    2. getAttribute("id")

      untuk melihat isi suatu attribute, contohnya melihat isi attribute id, return null jika tidak ada

    3. removeattribute("id")

      untuk menghapus suatu attribute, contohnya menghapus attribute id

  • classList

    cara khusus untuk memanipulasi class

    1. x.classList.add("newClass")

      untuk menambahkan class baru

    2. x.classList.remove("class")

      untuk menghapus class jika ditemukkan

    3. x.classList.toggle("class")

      jika class ditemukkan, maka hapus. Tapi jika tidak maka tambahkan, return true || false

    4. x.classList.item(0)

      untuk mengecek nama item class pada posisi tertentu, return namaClass || null

    5. x.classList.contains("class")

      untuk mengecek apakah ditemukkan class, return true || false

    6. x.classList.replace("class","newClass")

      untuk merubah suatu class menjadi class lain return true || false
