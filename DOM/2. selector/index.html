JavaScript

    OverAPI

  • Element

    Merupakan tag HTML seperti (h1-h6,p,div,dll)

  • Attribut

    Bagian dari element seperti (href,style,class,id,dll)

  • Text

    Nilai dari element html

  • Node

    1 Bagian dari HTML baik itu 1 element/ 1 value, intinya hanya 1

  • NodeList

    Kumpulan bagian dari HTML bebas element/ value, jumlahnya > 1

  • HTMLCollection

    Kumpulan element HTML (hanya element/tag HTML saja)

  • HTMLCollection and NodeList memiliki perbedaan cara pengaplikasian

  1. Root

    Node yang menjadi sumber dari semua node (document)

  2. Parent Node

    Node yang 1 tingkat diatas node lain, seperti body merupakan parent dari div

  3. Clild Node

    Node yang menjadi 1 tingkat dari node lain, seperti div merupakan child dari body

Method Penjelasan Hasil
getElementById() mengambil data berdasarkan id element
getElementsByTagName() mengambil data berdasarkan nama tag (seperti array) HTMLCollection
getElemetnsByClassName() mengambil data berdasarkan nama class (seperti array) HTMLCollection
getElemetnsByName() mengambil data berdasarkan attribute name biasasnya pada input form nodeList
querySelector() mengambil data seperti selector pada css (# untuk id, . untuk class) element
querySelectorAll() sama seperti diatasnya, tapi mengambil semua data nodeList
